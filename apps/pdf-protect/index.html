How to Use

  1. Click Download EXE and save SedgePDFProtector.exe to your computer.
  2. Double-click the file to run the app.
  3. Click Browse… and select the PDF you want to protect.
  4. Enter a single password (no email needed).
  5. Click Save… to choose where to save the protected PDF.
  6. Click Protect PDF to secure the file.

Your PDF will be encrypted with AES-256 and openable with your password in Adobe Reader, Edge, Chrome, or Preview.

Troubleshooting

  • “Failed to protect PDF” — Ensure the file is a valid .pdf.
  • PDF won’t open — Verify you’re using the exact password (case-sensitive).
  • SmartScreen warnings — Click More info → Run anyway or distribute a ZIP version to avoid browser flags.
