What it does

  • Protects a PDF with a single password (AES-256)
  • Works in Adobe Reader, Edge/Chrome, macOS Preview, etc.
  • Simple GUI: Browse PDF → Enter password → Save As → Protect

How to use

  1. Run SedgePDFProtector.exe
  2. Click Browse… and select your PDF
  3. Enter your password (toggle Show if needed)
  4. Click Save… to choose output (e.g. .protected.pdf)
  5. Click Protect PDF

Troubleshooting

  • “Failed to protect PDF” — Make sure the file is a valid .pdf.
  • Viewer won’t open — Ensure you typed the exact password (case-sensitive).
  • SmartScreen — Use a code-signed build for distribution at scale.
